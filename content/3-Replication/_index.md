#### Replication template

1. Access the AWS Management Console.

   - Find and select AWS Application Migration Service
   - Choose Get started

![Application Migration Service](/images/4/0001.png?featherlight=false&width=90pc)

Then, a default template will be created.

![Application Migration Service](/images/4/0002.png?featherlight=false&width=90pc)

2. You can customize the template.
